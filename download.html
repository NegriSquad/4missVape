Инструкция по установке

1

Подготовка среды

Установите Arduino IDE и добавьте поддержку ESP32 через менеджер плат:

Файл → Настройки → Дополнительные ссылки:
https://espressif.github.io/arduino-esp32/package_esp32_index.json
2

Установка библиотек

Установите необходимые библиотеки через менеджер библиотек:

  • Adafruit GFX Library
  • Adafruit SSD1306
  • Wire
3

Загрузка прошивки

Скачайте архив, распакуйте и откройте файл bruce_esp32.ino в Arduino IDE

4

Настройка и прошивка

В Tools выберите плату "ESP32 Dev Module", подключите устройство и нажмите Upload
